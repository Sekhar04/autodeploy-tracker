# 🚀 AutoDeploy Tracker

Enterprise-style CI/CD deployment tracking system built using Spring Boot, JWT Authentication, Role-Based Access Control (RBAC), MySQL, Docker, and REST APIs.

---

# 📌 Features

- JWT Authentication & Authorization
- Role-Based Access Control (ADMIN / DEVOPS / VIEWER)
- Deployment Lifecycle Tracking
- RESTful API Architecture
- MySQL Database Integration
- Dockerized Backend & Database
- Interactive Dashboard UI
- Secure API Endpoints
- Global Exception Handling
- Layered Backend Architecture
- Maven Build System
- JPA/Hibernate ORM

---

# 👥 User Roles

| Role | Permissions |
|------|-------------|
| ADMIN | Full Access |
| DEVOPS | Create + Update Deployments |
| VIEWER | Read-Only Access |
